[NGT0042] EcoHydro WorkFlow, version 1.0: R scripts for synthesizing data using metadata described with FRAMES

Author(s): Christoffersen B; Christianson D

Description: EcoHydro WorkFlow scripts synthesizes multiple data measurement types using metadata described in the NGEE Tropics FRAMES Metadata Templates v1.1. FRAMES provide a standardized format for recording metadata information necessary to interpret ecohydrological observations. EcoHydro WorkFlow reads and organizes metadata in the FRAMES Excel template files to enable metadata to be merged with data files, including raw data files. The R scripts are a series of modules. A README file provides instructions. Example data and FRAMES files are included. EcoHydro WorkFlow supports the measurement types, primarily from permanently installed sensors, described in FRAMES: dendrometry (tree stem increment), leaf temperature, sapflow, soil water content, and solar radiation. It also can accommodate meteorological data and can be extended to additional measurement types added in future. EcoHydro WorkFlow versions under development will be available at https://github.com/NGEET/metadata-tools/.
