[NGT0016] Sapflow, Weather - Veracruz Mexico, April 2015 to March 2016

Author(s): Berry C; Asbjornsen H


Dataset Information

Site ID: MX-CHI

Site Name: CHILES

Variables: Air Temperature; Precipitation (rainfall); Relative humidity; Sapflow; Soil water content (saturation); Solar Radiation; Wind speed/direction

Date Range: April 11, 2015 - March 1, 2016

Description: This data set includes sap flow, weather station, and one set of soil moisture data. It was collected for various other purposes within the scope of an NSF-ICER 1313804 project. There is sap flow and climate data across two sites from various points in time starting in the spring of 2015 and ending in spring of 2016. Sapflow data were collected using the heat ratio method (Burgess 2001). Sensors were installed at breast height and are constructed in Heidi Asbjornsen's lab at the University of New Hampshire. Each probe has either 2 or 3 thermocouples in them to measure flow at various depths and then multiplied by sapwood area at that depth. Corrections for spacing error and wounding were done like Burgess 2001. Data processing was done as in Berry (2016). I have also included one data set from the BSM site that has soil moisture data from last summer.

QA/QC: Provisional QA-QC

Methods Description: Some gaps in errors from the sap flow sensors have been filled using an ARIMA moving average model. Some trees have been corrected well for proper zero values while others have not, so this will need to be considered. Larger periods of errors have been removed entirely. Still lots of QC needed. For the weather data you will note some periodic gaps and a change of weather station type at the BSJ site.
