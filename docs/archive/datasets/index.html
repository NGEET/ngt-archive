NOTICE: We have migrated the NGEE-Tropics Data Archive to ESS-DIVE, the long-term data archive for BER/ESS. All public NGEE-Tropics datasets can be accessed in the new NGEE-Tropics Portal in ESS-DIVE, including after the NGEE-Tropics Data Archive is retired.

These datasets were originally published to be directly accessed only by the NGEE-Tropics Team. If you’re interested in accessing these data please contact the data contributors directly (contact information within each dataset). If you’re having difficulties contacting the data contributor, contact us at ngee-tropics-archive@lbl.gov.

This is a snapshot in time (July 2025) and will not be updated.
